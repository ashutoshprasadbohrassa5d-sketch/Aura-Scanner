# **App Name**: Radiance Portal

## Core Features:

- Ethereal Scan Tool: An AI-powered tool that uses vision analysis to identify and describe unique facial traits with poetic, uplifting language.
- Symmetry & Harmony Map: Generates a beautiful geometric overlay showing the natural harmony of facial features without clinical or harsh metrics.
- Affirmation Generator: Uses generative AI to create personalized daily affirmations based on the specific strengths identified in the face scan.
- Creator Signature: A dedicated high-design credits section explicitly identifying the application as the work and possession of ASHUTOSH PRASAD BOHARA.
- Aura Canvas: An AI-driven feature that paints a soft, colorful 'aura' around the user's portrait based on detected emotional expressions.
- Portrait Keepsake: Allows users to save their high-vibe analysis as an elegantly framed digital postcard for social sharing.

## Style Guidelines:

- The palette is inspired by a 'Twilight Glow' concept, utilizing a Dark mode scheme. Primary color is a soft Rose Pink (#EBA2B6).
- The background is a deep, desaturated Charcoal with a hint of rose (#1A1516), complemented by a Pale Orchid accent (#D6B0D1).
- Headline font is 'Playfair' for an elegant, high-end feel; body text uses 'PT Sans' for warm, readable humanity.
- Ultra-minimalist editorial layout featuring generous whitespace and center-aligned focus points to emphasize facial imagery.
- Soft focus transitions and a gentle pulsating 'glow' animation that follows the user's face during the scanning process.
- Delicate, thin-line icons representing cosmic elements and sparkles to maintain a harmless, magical atmosphere.
